# C-to-x86 LLM Compiler

Fine-tuned CodeLlama-7B to translate C code to x86 assembly using SFT, GRPO, and PPO.

- 48% compilation success rate with GRPO, outperforming GPT-4o-mini (32%)
- Dataset: https://huggingface.co/datasets/AzizZeynalli/c2x86-dataset
- Report: see compiler_report.pdf in this repo
